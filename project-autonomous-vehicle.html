Autonomous Vehicle with Smart Mobility

An embedded-systems-based autonomous vehicle concept integrating Arduino, PCB design, simulations, and mobile UI control.

Overview

This project extended the idea of a controlled vehicle into a more autonomous system. The goal was to design and simulate a platform that could make decisions with minimal human intervention, while connecting hardware design, simulations, and a user-facing interface.

What I Did

  • Designed and tested Arduino-based control logic for vehicle behaviour.
  • Worked on PCB design using Proteus to create a dual AC-DC power supply board.
  • Used TinkerCAD and related tools to simulate circuits and logic before implementation.
  • Contributed to mobile UI development allowing users to monitor or interact with the vehicle.

Tools & Technologies

  • Arduino microcontroller and C/C++-style embedded programming.
  • PCB design with Proteus for power and control circuits.
  • Simulation environments (e.g., TinkerCAD) for early validation.
  • Basic mobile UI components for command/control and feedback.

Challenges & Learnings

One of the key challenges was ensuring the designed PCB met both power requirements and safety margins while being compact and robust. This project improved my understanding of hardware-software co-design, and showed how important simulation is in catching issues before fabrication. It also gave me confidence to work at the intersection of electronics, code, and user experience.
